Documentacion de tarea- POO En estos ejercicios pude comprender mejor cómo trabajar con atributos, métodos, validaciones y diferentes estructuras en Java.

También noté que usar varias clases ayuda bastante a organizar el código y a detectar errores más fácilmente, ya que se puede ubicar rápidamente en qué ejercicio está el problema.

Se utilizaron atributos private para representar las características de cada objeto, como nombre, precio, autor, entre otros.

Para manejar estos datos, implementé métodos, algunos que devuelven valores, como en el caso de la calculadora, y otros que permiten realizar acciones, como aplicar descuentos o validar condiciones en distintos ejercicios, por ejemplo en los libros.

En general, utilicé métodos con return para obtener resultados y métodos void para modificar directamente los atributos de los objetos.

Para las validaciones, usé estructuras condicionales como if, lo que permitió evitar errores como la división entre cero y también impedir operaciones con valores negativos, como en el ejercicio de la cuenta.

Además, empleé el tipo de dato boolean para evaluar condiciones, como saber si un libro es largo o si una fecha es válida.

También utilicé constructores para inicializar los objetos con valores desde el momento en que se crean.

Finalmente, apliqué el encapsulamiento usando atributos private junto con getters y setters, lo que ayuda a tener un mejor control sobre los datos.
